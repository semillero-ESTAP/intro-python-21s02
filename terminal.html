Uso básico de la terminal

La terminal o consola es un ambiente/aplicación en el cual se pueden realizar acciones básicas y avanzadas de interacción con el computador como navegar a través de archivos, ejecutar programas o realizar configuraciones, todo basado en comandos de texto, a diferencia de como lo hacemos hoy en día basado en un interfaz grafica. Esta fue una de las primeras formas de interactuar con los sistemas operativos modernos.
Se ha mantenido vigente hoy en día, ya que permite entre otras cosas interactuar con programas que no requieren una interfaz gráfica como el interprete de Python, además de adaptarse de manera más óptima para algunos flujos de trabajo.

La terminal está incluida por defecto en culquier sistema operativo disponible hoy en día, para abrirla se busca la aplicación correspondiente:

  • Windows
    • PowerShell
  • macOS o Linux
    • Terminal

Commandos básicos del sistema

  • Mostrar el directorio actual:
    • pwd
  • Listar los archivos en el directorio actual:
    • ls
  • Cambiar de directorio o carpeta:
    • cd NombreDirectorio

Comandos básicos de Python en la terminal:

  • Abrir el interprete de python:
    • python
  • Ejecutar un script de python:
    • python archivo.py
  • Instalar un librería mediante PIP:
    • pip install nombrelibrería
  • Desinstalar un librería mediante PIP:
    • pip uninstall nombrelibrería
  • Listar las librerías instaladas:
    • pip list
