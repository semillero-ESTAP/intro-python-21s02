Instalación de Python

Es posible realizar la instalación de Python como cualquier otro pograma, basta de con descargar el instalador de los sitios oficiales. Actualmente existen dos metodos para relizar la instalación de python:

  1. Anaconda: Contiene todos los paquetes bases de python y algunas de las librerias mas usadas en ciencia de datos.
  2. Modular: Se instalan todos los elementos necesarios de manera individual el interprete o lenguaje y las librerias necesarias mediante un gestor de paquetes.
Por facilidad se hara la instalación mediante mediante Anaconda.

Instalación mediante Anaconda

Anaconda es una distribución de Python con interfaz gráfica enfocada a ciencia de datos, donde no solo se incluye la instalación de Python sino una gran cantidad de librerías enfocadas al análisis, visualización de datos y editores entre otros.

  • Windows
  • macOS

(Opcional) Instalación Modular

La instalación de Python se realiza como cualquier otro programa, en cualquiera de los sistemas operativos disponibles, tan solo basta con dirigirse al sitio oficial de Python descargar la ultima versión (actualmente 3.9.4) que corresponda con nuestro sistema operativo.

  • Windows
  • macOS
