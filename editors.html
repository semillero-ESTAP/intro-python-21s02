Editores de texto para Python

Python es un lenguaje interpretado lo cual implica que el código que escribamos debe pasar por un interprete el cuál ejecuta el código linea por linea sin la necesidad previa de ser compilado, como pasa en otros lenguajes. Para ello el código que sea escrito estará contenido en archivos con extensión .py, sin embargo, tambien existe la posibilidad de realizar la interpretación de estas lineas de código de manera interactiva y para ello se usaran archivos con extension .ipynb (interactive python notebook).

Jupyter Notebook

Jupyter es uno de los editores mas usados a la hora de trabajar con archivos interactivos de python (.ipynb), se trata de una aplicación basada en la web (se ejecuta en el navegador y de manera local), capaz de ejecutar codigo de Python ya sean archivos de texto plano (.py) o mediante notebooks interactivos (.ipynb). Este puede ser instalado mediante anaconda o usando el gestor de paquetes PIP.

  • Instalacion mediante PIP
    • pip install notebook

Visual Studio Code

Visual Studio Code (VScode) es un editor de código desarrollado por Microsoft que soporta gran variedad de lenguajes de programación entre ellos Python ya sea usando archivos de código plano (.py) o notebooks interactivos (.ipynb). Se caracteriza por ser un editor liviano compatible con una gran cantidad de extensiones que agregan gran cantidad de funcionalidades.

  • Windows
  • macOS
Para gozar de todas las funcionalidades de Python es VScode es necesario instalar dos extensiones: Python y Jupyter.
También es necesario tener una instalación activa de Python (Anaconda o Modular).

Google Colab

Google Colab es un editor en la nube similar a Jupyter, que puede ser usado por cualquiera y de manera que gratuita mediante una cuenta de Google. Una de las ventajas es que no es necesario configurar nada en el computador que estemos trabajando y los archivos o códigos desarrollados siempre estan accesibles en la nube.

  • Google Colab
