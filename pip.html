Gestión de librerías

En programación un librería es una extensión del lenguaje original con alguna finalidad especifica, por ejemplo en python existen librerías especificas para analizar y gráficar datos, otras librerías para realizar análisis estadísticos y aprendizaje automático, librerías para desarrollo web entre otros. Algunas de las librerías que serán usadas en esta serie de talleres son: pandas, matplotlib, seaborn y numpy.
Si se esta usando Anaconada como instalador, muchas de estas librerias ya vienen instaladas por defecto, sin embargo si se usa una instalación modular o si desea instalar paquetes adicionales que no esten incluidos con Anaconada es necesario usar un gestor de paquetes externos como lo es PIP.

PIP

Para tal fin python cuenta con herramientas integradas (se incluye por defecto en la instalación) para la gestión de librerías (instalar, desinstalar y actualizar) una de ellas es PIP, una herramienta que se usa mediante la consola o terminal.

Comandos básicos de PIP
  • Instalar librería
    • pip install NombreLibreria
  • Desinstalar librería
    • pip uninstall NombreLibreria
  • Actualizar librería
    • pip install --upgrade NombreLibreria
